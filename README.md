# NULL Base Filesystem
This repo serves the base files for bootsrapping NULL using `pkglet`. Specifically the `null.basefs` package on the `main` repo.

## Note
This package should NOT be used outside of `null.basefs`, unless you know what you're doing.. of course.
You can look at the `null.basefs` package manifest to see where files go.
